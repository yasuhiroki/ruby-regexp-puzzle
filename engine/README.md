# 正規表現エンジンを作ってみる

## first.rb

己の頭の中から深く考えず実装を進めてみたパターン。  
`*` と `|` のみ対応。  
継承して処理できる記述方法を増やしていくパターン。

パターンが増えれば増えるほど継承の階層が増えていく。
