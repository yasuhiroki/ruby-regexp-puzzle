# 概要

ハッシュタグをパースする正規表現

## `\s` を使うと全角空白が残る

```bash
$ cat text | ruby -ne 'p $_.scan(/#[^#\s]+/)'
["#タグ", "#タグ", "#タグ", "#たぐ", "#tag", "#たぐ"]
["#たぐ", "#タグ　", "#たぐ", "#🐧"]
```

## `[:space:]` を使えば全角空白を取り除ける

```bash
$ cat text | ruby -ne 'p $_.scan(/#[^#[:space:]]+/)'
["#タグ", "#タグ", "#タグ", "#たぐ", "#tag", "#たぐ"]
["#たぐ", "#タグ", "#たぐ", "#🐧"]
```
